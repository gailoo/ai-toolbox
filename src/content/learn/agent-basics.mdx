## Agent 和 Chatbot 的区别

Chatbot 主要负责回答问题；Agent 不只是回答，还会拆解任务、调用工具、执行操作并根据结果继续迭代。

## 一个 Agent 的核心组成

- **模型**：负责理解、规划和生成。
- **工具**：搜索、文件、数据库、浏览器、API 等外部能力。
- **记忆**：保存长期偏好、上下文和任务状态。
- **规划**：把复杂目标拆成步骤。
- **反馈循环**：执行后读取结果，再决定下一步。

## 常见 Agent 场景

- 自动生成市场周报。
- 监控社媒和竞品动态。
- 自动写代码、跑测试、修复错误。
- 企业内部流程自动化。

## 风险与边界

Agent 的风险来自“能行动”。所以要给它明确边界：

1. 外部发送、删除、支付等动作必须确认。
2. 对生产系统操作要有权限和审计。
3. 关键任务要设置可回滚机制。
4. 不允许模型凭空生成业务事实。
